<Autobiography>

自傳

喜歡動手實作且樂於接受挑戰，遇到未曾碰過的技術總會積極應對，並將所學的新知以簡單易懂的方式筆記下來，期望能在幾年後技術更成熟且筆記內容更完整時，放上網路作為技術文章供初學者網頁前端全方位的學習。關注在設計方面是自己的日常興趣，常使用 pinterest 與 behence 蒐集各類型的設計點子，提升自己平時對於美感的標準。近期在網頁前端部分較多以熟練切版與網頁互動效果為目標，另外並行學習串接和 React 框架，期望在未來的一年內將 React 框架基本應用學會並熟練實作，讓自己有獨立完成專案的能力。也期望工作上與同事共事的環境下，在設計師與後端工程師兩方面的職務內容也積極撥空花費時間學習了解，建立與設計師和後端工程師有效對話的溝通橋樑。


因注重跨領域的學習，大學期間主動參與多樣的課程、活動與計畫。課程方面包含行銷管理、藝術與美學講座等等，除了奠定基礎外，也能促進發掘與本科系(資訊傳播)的專業結合的可能性。 另外自己雖為科技組別，較注重於程式方面的撰寫開發，同時也積極修習設計組別的設計相關課程，同一時期因參與其他院所舉辦的額外課程，老師對於非本科系的自己積極的學習態度留下的深刻印象，推薦我擔任手冊編輯一職，因此我也接下這項任務，增加自己對於不同專案的實務經驗。
